O começo do HTML


        O HTML foi criado em 1990-1991 por Tim Berners-Lee no CERN para compartilhar documentos de pesquisa, evoluindo de uma marcação simples de 18 tags para a base da web moderna. A evolução passou pelo HTML 2.0 (1995), HTML 4 (1999) e culminou no HTML5 (2014), que trouxe semântica, multimídia nativa (áudio/vídeo) e APIs avançadas.





        O HTML é uma linguagem de marcação utilizada na construção de páginas na web. Ou seja, o HTML não é uma linguagem de programação em que o programador tem liberdade de programar comportamentos a partir da entrada de dados. Em vez disso, as linguagens de marcação descrevem informações de apresentação para que possam ser apresentadas aos usuários. O HTML é hoje utilizado por todo e qualquer site na internet para a criação e estruturação do seu conteúdo.
        Os navegadores (Firefox, Chrome, Safari, etc) são programas responsáveis por interpretar o código HTML e transformá-los em páginas visuais para que os usuários possam interagir.



      HTML 2.0:

  Primeira versão padronizada do HTML (1995), focada em estrutura básica de páginas com textos, links e formulários simples. Não possuía muitos recursos de layout ou multimídia.

      HTML 4:

  Lançado em 1997, trouxe maior suporte a estilos (integração com CSS), scripts e melhor estruturação do conteúdo, separando apresentação e estrutura.

      HTML 5:

  Versão moderna do HTML (2014), adicionou suporte nativo a áudio, vídeo, gráficos (canvas), APIs e melhor semântica para criar aplicações web mais ricas sem depender de plugins.


        Para finalizar, a história do HTML mostra como a web evoluiu ao longo do tempo, passando de páginas simples de texto para sites modernos, interativos e cheios de recursos. Cada versão do HTML contribuiu para tornar a internet mais acessível, organizada e poderosa, sendo até hoje uma das principais bases para a criação de páginas e aplicações na web.
